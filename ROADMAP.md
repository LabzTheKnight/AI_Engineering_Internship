    - authentication
    - tests
    - Docker Compose
    - GitHub Actions CI

Phase 2 --Basic LLM Apps:
    Build an AI study assistant:
    - user enters notes
    - app summarizes them
    - app generates quiz questions
    - app stores history
    - add prompt templates
    - add error handling

Phase 3 RAG
    Build a document Q&A system:
    - upload PDFs/text
    - chunk documents
    - create embeddings
    - store vectors
    - retrieve relevant chunks
    - answer with citations

Phase 4: Agents
    Goal: build tool-using AI systems.
    Build an agent that can:
    - search local documents
    - call a calculator tool
    - query a database
    - decide which tool to use
    - log every step safely

Phase 5: Evaluation
    Goal: stop just "vibe testing".
    Create an eval suite:
    - test answer correctness
    - test hallucination rate
    - test retrieval quality
    - test latency
    - compare prompts/models
